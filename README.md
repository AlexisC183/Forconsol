# Fraccion Estructura
Ejemplos de formatos de Fraccion válidos (métodos Fraccion.Parse(String) y Fraccion.TryParse(String, Fraccion)):
- `"15"`
- `"15 / 11"`
- `" 15 / 11 "`
- `"-15/11"`
- `"15 /-11"`
- `"-15/   -11"`
